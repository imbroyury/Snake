# Snake
Classic Snake Game

## How To Play:
* Enter username
* Hit Spacebar
* Use ← ↑ → ↓ or WASD to control the snake

## Features:
* Snake tail is colored as per last food piece eaten
* Leadeboard!

## Build using: 
* HTML5 Canvas
* RequestAnimationFrame
* LocalStorage
* [randomColor library](https://github.com/davidmerfield/randomColor)
